 11. Consider the following code:

    function dateTodayTomorrow(a, b) {
      a.concat(b);
      return a.valueOf();
     }
    alert(dateTodayTomorrow(“Wednesday”, “Thursday”)); 

    What message will appear in the alert box when this script 
    is run in the browser?

    a. Wednesday
    b. ThursdayWednesday
    c. Thursday
    d. WednesdayThursday
       


Equals C 

  
  function dateTodayTomorrow(a, b) {
    c = a.concat(b);
    return c.valueOf();
    }
  document.write(dateTodayTomorrow("Wednesday", "Thursday"));
   


b.valueOf();
